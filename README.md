# 8-BIT-ALU

*Company* : Coddec technologies

*Name* : k.likhitha

*Domain* : VLSI and Digital electronics

*Duration* : 4 weeks

This project implements an 8-bit low-power ALU using CMOS logic and clock gating techniques to reduce dynamic power consumption.
The design supports arithmetic and logical operations and is verified through Verilog simulation and synthesis tools.
Layout fabrication and validation are performed using industry-standard VLSI tools to analyze area, delay, and power performance.
